# Datvida LLC Website

A clean static website ready for Netlify, Vercel, Cloudflare Pages, or any simple hosting.

## Files

- `index.html` — main website
- `styles.css` — design/style
- `netlify.toml` — Netlify config
- `README_DEPLOY.md` — deployment steps

## What to edit first

Open `index.html` and replace:

- `+1 (000) 000-0000`
- `info@yourdomain.com`
- `yourdomain.com`
- location/services if needed

## Netlify deploy

1. Create a new GitHub repo, for example: `en-trade-logistics-site`
2. Upload these files.
3. Go to Netlify → Add new site → Import from GitHub.
4. Select the repo.
5. Build command: leave empty.
6. Publish directory: `.`
7. Deploy.
8. Add your GoDaddy domain in Netlify domain settings.
